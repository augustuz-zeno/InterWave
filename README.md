# 🧬 Procedural PFP Generator

Генератор уникальных аватарок на основе никнейма. Использует NumPy для быстрой генерации математических узоров с эффектами постобработки.

## ✨ Особенности
- **Детерминированность**: Один ник — один и тот же результат.
- **Эффекты**: Хроматическая аберрация, зернистость и виньетка.
- **Производительность**: Векторизация через NumPy.
